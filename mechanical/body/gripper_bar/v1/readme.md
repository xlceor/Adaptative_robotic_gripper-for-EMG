# Linkage bar v1

## Purpose
link base that connects the TPU finger to the base via the mechanism.

## Geometry
The shape was designed to reduce weight while maintaining strength. 

## Material
Currently PLA. In future versions, the material will be remplaced with PETG

## Comments of the version
This model is the first to be tested under real-world conditions. Unlike the previous version, the center-to-center distance is greater for increased gripping capacity. Additionally, the number of screw holes has been increased to two, eliminating the problem of the previous version (where the head rotated using the center screw as an axis). Furthermore, the head geometry has been adapted to reduce weight while maintaining strength.

## Next Steps
Keep testing to see the response of the material under weight and load.
