# Geared bars v3

## Purpose
principal bars of the mechanism. Translates the servo output to the open/closing mechaism of the gripper. 

## Geometry
The geometry was simplified to test the new version. Unlike previous versions, the transmission is located in the rear area. So, while mantaining the same geometry, it can provide a bigger range of motion. also, the module and the number of teeth was adjusted to the new distance between center point

## Material
PLA for testing

## Comments of the version
The reduction of the servo output was lower than before due to design limitations. The actual reduction rate is 20/17. In the project, more torque is required, so the geometry will we adapted to manage more reduction.
