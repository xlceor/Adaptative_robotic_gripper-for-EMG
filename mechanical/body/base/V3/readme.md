# Base v3

## Purpose
Base of the four-bar linkage. It serves as a support and for storing the electronics, including the servos and MCU.

## Geometry
The shape was designed in base of the structure of 2 servos tandem-mounted and the module Xl4016.

## Material
PLA for testing

## Comments of the version
The total torque was insufficient for the aplication needed. So, the design was remodeled based on 2 servos working together, hopefully multiplying the torque by two.
