# TPU Finger Prototype v1.1

## Purpose
TPU finger of adaptable soft-robotic gripper. 

## Geometry
Right triangle base with parallel internal bending lines to allow controlled deformation.

## Material
TPU 95A

## Comments of the version
The base model still the same that the previous version, but the external wall -the one that is in contact with the object- now has texture, adding friction and gripping capacity. Even so, the thickness of the external wall was small.
## Next Steps
Test under real-life load to meassure the resistance of the finger.
